## 📖 Acerca del Proyecto

**VeloGrid** es una aplicación diseñada para facilitar la movilidad en bicicleta dentro del entorno urbano. El sistema permite la gestión, planificación y optimización de rutas, brindando a los ciclistas herramientas útiles para mejorar su experiencia de transporte diario. 

Este proyecto aplica los patrones de diseño, principios de arquitectura de software y buenas prácticas de codificación aprendidos durante el curso de Programación III.

### ✨ Características Principales
* **Gestión de Rutas:** Creación, edición y visualización de trayectos seguros para ciclistas.
* **Interfaz Intuitiva:** Diseño pensado en la usabilidad y la rápida interacción.
